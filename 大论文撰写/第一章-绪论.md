## 1.1 研究背景与意义【待增加】
### 1.1.1 研究背景
据第七次全国人口普查结果显示，中国60岁及以上人口占比超过18%，成为世界老年人口最多的国家[1]，数据预测中国将会在2030年进入深度老龄化社会。随着时代和社会进步，中国传统家庭结构发生改变，子女独居成为常态，因此近几年社会中出现新的弱势群体“空巢老人”。人口普查的资料说明我国的独居老人呈持续上升趋势，尤其在一线大城市，空巢老人比例高达30%~50%，独居老人的人身安全和有效监护问题已经成为近几年公共卫生保健邻域的重要挑战和学术研究热点。在这快速老龄化的社会，老年人的健康问题日益凸显，失能、半失能老人数量急剧增加许多老年人具有多种并发症，跌倒成为老年人生命安全的重要威胁之一，据中国疾控检测系统调查结果显示跌倒已成为中国65岁及以上老年人因伤致死的首要原因[2]，跌倒后若能得到即时救治则可以降低80%的死亡率[3]。大部分年轻人无法时刻陪伴父母，导致一些老年人独自在家跌倒后存在无人发现致死的风险。随着家用摄像机清晰度增加及设备的不断优化，许多年轻人选择使用家用摄像头对老年人的人身安全进行实时检测和预警，以实现第一时间预判独居老人跌倒创伤行为并降低老年人因跌倒产生的生命风险。【增加疾病导致跌倒行为、心脏骤停、中风、老年痴呆等疾病研究统计】
### 1.1.2 研究意义 【优化】
卫生部公布的《老年人跌倒干预技术指南》中指出“跌倒是我国65岁以上老年人伤害死亡的首位原因”意外跌倒是许多社区的主要公共卫生问题。对监控视频下的跌倒行为动态研究，实时识别和预测监控视频中的跌倒行为是智能视频监控系统的一个重要组成部分。跌倒行为是人体行为的重要组成部分之一，人体行为检测技术主要有三个主要研究邻域：第一个研究领域是基于传感器的人体行为检测方向，主要采用穿戴式的加速度计或手机陀螺仪进行人体生物特征信息提取再进行人体行为分类归类，但是该类传感器不适合老年人群体携带和使用。基于环境传感器的室内声波的传感器对于环境要求较高，容易受到外接因素的干扰。压力传感器可以根据步态信息分析老年人异常步态并判断跌倒行为，但其应用性较差无法普及家庭使用。第二个研究邻域是基于物联网智能家居的人体行为检测，根据智能家居连结云端读取老年人使用智能家居的数据进行异常判断但该研究邻域缺乏隐私保护。第三个研究邻域是基于计算机视觉的人体行为检测方法，家庭单目监控摄像机因其物美价廉的特点成为许多家庭选择完成空巢老人监护的设备之一，计算机视觉算法的不断创新为人体行为检测带来许多便捷和突破。随着科技的不断进步及监控设备的更新换代，家庭监控装置逐渐进入大家的视野，为了保证家庭成员特别是老年人的身体健康，例如独居老人跌倒、晕倒等突发情况，在监控装置中设置智能视频监控系统及时检测和报警尤为重要；目前，意外跌倒是影响老年人安全健康和生活质量的重要因素，人口老龄化趋势意味着将有更多老年人面临意外跌倒致病的威胁。由于大部分成年人无法随时看护老人，准确、迅速、及时地检测跌倒行为对于保障老人健康至关重要。因此，对人体动态分析后进行跌倒行为的识别预估并将其应用嵌入到智能视频监控系统中尤为重要。基于以上分析，需要设计一种适合独居老年人和大部分室内单目监控设备的精确轻量级的跌倒异常行为检测模型，同时兼具隐私保护特点。目前国内外基于计算机视觉的跌倒检测方法主要关注视频数据自身进行特征提取和算法分类识别检测，针对视频数据特征序列关系下轻量级和隐私保护的嵌入式跌倒检测算法研究较少。本文不仅提出【】模型进行动态前景目标作为数据分析和跌倒行为特征研究分析的依据，而且创新性提出新的跌倒行为特征提取方法，兼具算法隐私保护和特征精准提取的要求，同时本文提出【】轻量化算法对特征信息进行分类预测。轻量级的嵌入式算法的研究虽然难度较大，但目前国内外许多研究者已经开始进行研究。相信在国内外研究者的不断分析研究中一定会在未来产出更多更有价值的跌倒检测算法并推动其发展创新并应用于家用监控设备中，为推动人口老龄化社会卫生安全发展做出贡献。
## 1.2 国内外研究现状 【补全】
跌倒行为检测是医疗和智能家居系统的研究重点之一，迅速高效的跌倒检测算法可以极大减少跌倒对老年人的伤害，迅速识别预警可以提升老年人因跌倒引起其他并发症的存活率。跌倒检测技术主要分为基于传感器的信号识别技术、基于物联网云技术的智能家居检测技术及基于计算机视觉的跌倒识别技术，以下对在室内场景下跌倒识别技术研究现状的阐述。下表是对不同跌倒检测方式的对比【表1】【模仿综述文章】
### 1.2.1 基于传感器的跌倒检测技术
基于传感器的跌倒检测方法主要是穿戴式传感器和环境传感器两种方式。【Cuong】[4 Combining skeleton and accelerometer data for human fine-grained activity recognition and abnormal behaviour detection with deep temporal convolutional networks 2021]等人提出一种通过融合组合骨架和加速度数据的多苷脂模态框架用于人体细粒度识别和异常行为检测。该框架由扩展的因果卷积分量组成的深度时间卷积网络（Temporal Convolutional Network TCN）来进行特征学习和处理时间特性。在TCN中用卷积层学习和表示的特征图被馈送到两个完全连接的层中用于预测。该框架由加速度信号预处理、时间卷积网络(TCN)特征提取和融合特征骨架和加速度特征信息三个部分组成。由于该框架的数据和设备的要求，框架需要结合监控设备和加速度传感器采集人体行为特征信息且实验数据具有局限性且加速度特征指标可以使用数学物理公式方法在视频数据中进行特征转化，因此该框架在现实环境中准确性和实用性较差。【Kumar】[5 Fall Detection and Activity Recognition using Hybrid Convolution Neural Network and Extreme Gradient Boosting classifier 2022]等人提出一种基于腰间惯性传感器设备的跌倒行为分类检测方法，算法使用支持向量机(Support Vector Machine SVM)对腰间惯性数据记录中的每一条信号数据计算加速成分的均方根值，然后使用卷积神经网络(Convolutional Neural Network CNN)构建卷积过滤层和平均下采样层对归一化信号数据进行特征提取，最后使用XGB分类器进行信号正常人体行为和跌倒行为二分类工作。该算法虽然具有轻量化特性但是仅使用腰间惯性数据作为跌倒检测依据容易造成将类跌倒行为误判问题，例如将人体躺下、蹲下等惯性变化过程误判为跌倒行为的问题。【Uzoma】[6 Smartphone Motion Sensor-Based Complex Human Activity Identification Using Deep Stacked Autoencoder Algorithm for Enhanced Smart Healthcare System 2020]等人提出一种基于智能手机运动传感器的复杂人体活动识别系统，该方法利用目前年轻人使用广泛的智能手环设备中嵌入式陀螺仪和人体健康检测传感器进行特征信息提取，提出一种深度堆叠自动编码器方法实现人体跌倒和异常行为监测。首先，对智能手环中陀螺仪数据与3D加速度计数据进行融合特征综合评估分类。其次，对基于人体异常行为的角度和加速度物理学特点和医学心率异常变化特征分别对陀螺仪加速度计数据和人体机能数据进行多特征堆叠卷积神经网络特征提取。最后，使用自编码分类器对特征信息异常分类检测。该方法将人体医疗理论基础和物理学理论相结合提升人体异常行为检测的准确性，但智能手环在老年人生活中使用不具广泛性且信息处理过程较复杂实时性较差。
从上述归纳中可以看出，随着传感器技术的更新换代，穿戴式传感器的功能和人体机能特征提取能力不断增强，可以收集更多受试者的运动信息和健康信息，例如人体运动速率、运动偏移角度、相对旋转角度和心率变化等，可以从人体指标变化方面捕获更多跌倒行为对身体的影响，因此穿戴式传感器在人体医疗研究邻域应用广泛。在现实生活中，穿戴式传感器的异物感导致老年人无法长时间佩戴，并且老年人对智能手环需求较低，记忆力下降导致忘记佩戴的问题导致穿戴式传感器不适合老年人跌倒行为检测研究。
【Li】[7 A Heterogeneous Ensemble Learning-Based Acoustic Fall Detection Method for Elderly People in Indoor Environment 2020]等人提出一种基于室内声学跌倒检测的异构集成学习方法(heterogenous ensemble learning HEL)。该方法通过声音传感器收集室内环境的坠落声音并通过提取短时能量信息获取特征和图谱信号，使用基于k近邻分类器(KNN)、支持向量机和卷积神经网络的异构集合分类模型对特征信息进行二分类，即对声音信号特征数据分类为跌倒行为和非跌倒行为。声音传感器对环境要求较高，一般在实验环境中控制其他变量的情况下才可以达到较好的跌倒检测精准性。【Zhao】[8 Distributed Radar-based Human Activity Recognition using Vision Transformer and CNNs 2021]等人提出一种基于多分布式超宽带雷达系统的跌倒行为检测分类框架。该框架使用5个分布式雷达提取受试者沿特定轨迹执行9个指定动作的距离多普勒图像，并基于多头注意力机制分类器对特征信息进行二分类。该分类器是一种Transformer架构，对输入的多普勒图像根据时间序列关联性特征信息进行跌倒行为和非跌倒行为的二分类代替卷积神经网络通过特征标签信息更迭训练。
环境传感器采集的数据大部分以波频数据为主，环境中细微的变量差异往往会产生不同的检查结果和性能。因此基于环境传感器的人体行为和跌倒行为检测在实验室研究场景及简单人体行为运动检测场景中有较好的准确性。现实场景存在较多环境干扰因素，例如环境杂音、噪声等，基于波频信号的跌倒行为特征提取和去噪降噪方法因场景差异无法达成统一性和鲁棒性，同时在预处理波频阶段可能会造成削弱目标特征频率数据信息，从而导致基于环境传感器的跌倒检测算法准确性下降的问题。
### 1.2.2 基于智能家居的跌倒检测技术
随着人工智能和云技术的不断发展，部分家庭出现了智能家居辅助陪伴老年人的日常生活。
【物联网智能家居的识别技术】【Cheng】[9 A Device-free Human Fall Detection System Based on GMM-HMM Using WiFi Signals 2021]等人提出一种基于wifi设备的高斯混合模型隐马尔可夫模型(GMM-HMM)跌倒行为检测方法。该方法通过部署多wifi设备中的物理层CSI提取人体目标香味差矩阵作为跌倒行为检测依据，设计一种利用滑动窗口和标记方法拦截跌倒行为信号的方法代替分类模型，GMM-HMM模型使用前向-后向算法输出活动类别的概率分布，并根据最大概率观测值推断人类活动从而推断是否发生跌倒。该方法可以根据室内人体目标活动情况实时检测异常信号波动，是一种非接触式跌倒检测识别方法。
【王宇】[10 室内环境下老年人异常行为检测 2022]等人提出一种基于步态生物学压力技术的智能地毯设备，该设备通过嵌入地毯内的压力传感器采集人体目标步态特征信息作为人体行为运动的关键信息，通过对正常步态和异常步态的差异分析使用基于支持向量机分类算法结合步态特征和压力强度数据进行跌倒检测识别，使用物联网信号实现实时报警。
目前，基于智能家居的跌倒检测技术在实验室场景场景下检测效能较好，智能家居的器材消耗大且造价高是影响了智能家居领域实时跌倒领域的因素之一。同时，智能家居的数据计算一般存储在其所属的云端，云技术的应用可以在短时间内提供更多资源和算力，在一定程度上提升了跌倒检测算法的准确性，但是受试者的个人隐私安全方面将受到一定的影响。
### 1.2.3 基于计算机视觉的跌倒检测技术
随着摄像设备的不断更新换代，家用监控摄像机越来越普及。【Merrouche】[11 Fall detection based on shape deformation 2020]等人提出一种基于目标形状变形和运动信息的跌倒行为分类检测方法，该方法使用视频帧间图像剪影变形构造新的特征描述符，并提出基于时间序列视频图像序列步骤步态行为姿势间的差异变化，二者共同作为人体动作分类的判别信息完成随机森林、Adoboost和KNN算法的交叉验证实验。该方法仅使用视频帧间差异构建人体目标形变序列作为分类算法的单一特征信息，未考虑跌倒行为比人体目标其他日常行为更复杂，并且该文章使用单一数据集作为研究对象降低跌倒行为检测算法的鲁棒性。【Ke】[12 Temporal action detection based on two-stream You Only Look Once network for elderly care service robot 2021]等人提出一种融合You Only Look Once(YOLO)网络模型和关节点骨架模型的跌倒检测模型，分别使用YOLO算法识别人体目标对象以及关节点骨架特征信息，将数据分割为视频信息流和骨架流进行分网络训练，通过数据编码转换方法将时空时间动作检测转化为不断增强的特征空间一维检测对象，最后使用层融合实现管道融合。该过程需要对实验数据进行精细的预处理、降噪和分类来提升模型跌倒检测的精准度，模型对于人体目标行为的运动过程和轨迹较为敏感，在人体类跌倒行为的检测中性能较差。【谭荣】[13 视觉隐私保护下室内监控视频中的跌倒行为检测研究 2021]等人提出了基于视频压缩感知技术的监控视频跌倒行为检测方法，该方法根据多层压缩感知的视觉屏蔽感知编码模型对监控视频进行隐私保护处理，对高保真压缩图像提取三维纹理特征来区别日常人体行为和跌倒行为。同时，该方法引入生成对抗网络(generative  adversarial network, GAN)构建私有信息嵌入模型识别特征信息差异并归类。该方法在视频隐私保护上提出创新性方法，但对视频分辨率和清晰度提出更高的要求才能在后续图像纹理特征提取和处理阶段达到较好的检测分类性能效果。【马敬奇】[14 基于AlphaPose优化模型的老人跌倒行为检测算法 2022]等人提出一种基于AlphaPose关键点骨架模型的老人异常行为检测算法，通过人体姿态关节点数据提取跌倒行为中变化突出的关节点组合信息并进行坐标数据转换，最终使用阈值判断人体目标跌倒瞬间特征点数据信息结果达到跌倒检测目的。该方法将图像信息转化为物理特征信息，例如目标运动速度，躯干间角度点信息，简化了图像信息在算法中训练的消耗实现算法低功耗，但是多固定阈值判定跌倒行为容易收到人体其他行为干扰造成误判问题。【Chaudhari】[15 Towards a Template Matching Approach for Human Fall Detection 2021]等人提出一种基于跌倒模板匹配的单目监控视频跌倒行为匹配算法，该方法基于目标特征信息匹配策略从实时视频帧中提取前景动态目标模板覆盖历史受试者运动区域构建历史人体目标模板，然后分别创建正常人体行为模板和跌倒异常行为模板，基于KNN算法对待测视频模板匹配差异进行跌倒和飞跌倒行为标签处理，最后遍历标签利用概率统计方法判断该视频是否为跌倒行为。该方法适用于完整的视频记录信息，在实时视频跌倒判断过程中出现一定延时处理问题。
监控视频数据的井喷式增加可以加大提升基于计算机视觉的跌倒检测技术的准确性，且具有更为丰富的特征提取方法进一步提升跌倒检测方法的多维性。近年来，深度学习算法轻量化研究为算法嵌入监控系统带来新的方向。

为阈值分析法、基于机器学习的检测算法和基于深度学
习的检测算法三类。阈值分析法是将实时的加速度或
二次处理后的均方值等数据作为特征值，并与通过大量
跌倒实验得出的阈值作比较，当数据值超过或低于阈值
时判定为异常情况。基于机器学习的检测算法则是通
过人工设计的特征算子来处理并描述底层视觉数据，比
如 HOG[4]
特征、ORB[5]
特征和 LBP[6]
特征等，该类算法含
有多种功能的分类器，具有显著的分类优势。但是该类
算法过度依赖于人工提取动作特征，一旦特征样本发现
偏差，跌倒检测的性能便会极度下降，并且上述的特征
算子难以获取复杂图像中的语义信息，而基于深度学习
的检测算法能较好地弥补的该缺点，可以自动获取样本
数据中的特征值，形成一种学习策略，拥有很好的特征
表示能力。】
